# Module registry

EveSite Forge v0.1 ships with three interactive modes:

- `none` — document-first interface;
- `weighted-risk` — weighted controls, interactions, thresholds, event state, and attribution;
- `maturity-ladder` — staged capability and governance model.

Future modules should expose:

- a stable module identifier;
- accepted schema fields;
- rendered public meaning;
- limitations and evidence requirements;
- a fallback representation for non-interactive environments.
