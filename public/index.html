System

不是一個固定模板，而是一組網站生成與治理層。

共同骨架處理可重用部分；網站類型保存領域差異；互動模組承載真正需要被操作的模型。

Governance

生成很便宜，一致性與誠實性才是新的成本。

每個公開介面都必須能指出正式來源、證據等級、適用邊界、版本狀態與尚未完成之處。

Public ≠ Formal門面是投影，不是正式理論的替代品。
Interactive ≠ Proven可操作展示不等於模型已被實證支持。
Generated ≠ Governed能生成網站，不代表已處理版本、責任與漂移。
Shared ≠ Identical共同 Schema 不應抹平不同領域的特殊表達。
