# GWSM v0.1 — Limitations & Unknowns

This document records the current boundary of the Game Wave Semantic Model demonstration.

## Current limitations

- The interactive model is a conceptual demonstration, not a calibrated game simulation.
- The parameters are illustrative and have not been validated against a shipped game.
- Event attribution is simplified and does not establish real-world causality.
- The current runtime is static browser-side JavaScript and does not persist world history.
- AI adaptation remains a governed roadmap item rather than an enabled autonomous function.

## Unknowns

- Which wave channels remain interpretable at large world scale.
- How much coupling complexity designers can manage before visual tooling becomes necessary.
- Which stability constraints are sufficient across different genres.
- How to compare designer-authored dynamics with learned dynamics without losing authority and provenance.

## Non-claims

GWSM v0.1 does not claim that every game state should be represented as a wave, nor that wave-based representation automatically improves gameplay.
