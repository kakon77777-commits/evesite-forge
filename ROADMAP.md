# EveSite Forge Roadmap

## 0.2 — Site type separation

- Extract common page, site, and site-group schemas.
- Add product, documentation, observatory, and catalog site types.
- Introduce module contracts and fallback rendering.

## 0.3 — Content import

- Markdown front matter import.
- Document-to-schema assistance.
- Formal/public layer difference reports.
- Version drift detection.

## 0.4 — Multi-site governance

- Shared navigation and catalog generation.
- Multilingual routing and terminology registries.
- Absolute URL and canonical metadata generation.
- Deployment profiles for GitHub Pages and Cloudflare Pages.

## 1.0 — Governed universal generator

- Stable schema versioning.
- Reusable interaction module SDK.
- Evidence, limitation, provenance, and authority policies.
- Batch generation for an evolving website network.
